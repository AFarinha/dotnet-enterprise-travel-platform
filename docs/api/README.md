# API Documentation

OpenAPI is available from the Gateway API at:

```text
/openapi/v1.json
```

Trips endpoints implemented in Phase 1:

- `POST /api/trips`
- `GET /api/trips/{id}`
- `GET /api/trips`
- `POST /api/trips/{id}/travellers`
- `POST /api/trips/{id}/plan`
- `POST /api/trips/{id}/cancel`
- `GET /api/trips/{id}/timeline`
